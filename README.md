Arki Gas — Plataforma de Compra de Gás Online

O Arki Gas é um website que conecta empresas fornecedoras de gás com clientes finais, permitindo a compra de gás de forma rápida, simples e segura.
Este projecto foi criado com o objetivo de facilitar pedidos de gás doméstico através da internet.

FUNCIONALIDADES

- Visualizar produtos de gás disponíveis
- Comprar gás online
- Interface simples e moderna
- Preparado para deploy no Vercel
- Projecto pronto para evolução futura (login, pagamentos, entregas)

 TECNOLOGIAS UTILIZADAS

- HTML5
- CSS3
- JavaScript
- GitHub (controlo de versão)
- Vercel (deploy)
